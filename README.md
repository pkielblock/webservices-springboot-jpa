## Web Services - Springboot and JPA
Project to study Web Services with Springboot and JPA (Hibernate)

This project was created to study:

* Java Springboot
* Rest API
* Domain Model
* Structure Logical Layers (resource, service, repository)
* JPA
* Hibernate
* CRUD
* Maven
* H2 Database
* Handling Exceptions
