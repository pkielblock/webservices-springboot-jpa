## Web Services - Spring Boot and JPA
Project to study Web Services with Spring Boot and JPA (Hibernate)

This project was created to study:

* Java Spring Boot
* Rest API
* Domain Model
* Structure Logical Layers (resource, service, repository)
* JPA
* Hibernate
* CRUD
* Maven
* H2 Database
* Handling Exceptions
